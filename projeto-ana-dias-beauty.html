Ana Dias Beauty

Sistema web completo para salão de beleza

Sobre o Projeto

O Ana Dias Beauty é uma aplicação web desenvolvida para gerenciar um salão de beleza fictício, oferecendo uma experiência completa tanto para administradores quanto para clientes.

O sistema foi desenvolvido com foco na usabilidade e design responsivo, garantindo uma experiência fluida em todos os dispositivos. A aplicação inclui um catálogo de serviços, galeria de trabalhos e área de contato fictícia.

Funcionalidades Principais

  • Plataforma de e-commerce bonita e minimalista
  • Catálogo de serviços
  • Galeria de trabalhos realizados
  • Design totalmente responsivo
  • Formulário de contato
