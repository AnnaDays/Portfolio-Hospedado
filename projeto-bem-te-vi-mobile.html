Bem Te Vi Pet-Shop (Mobile)

Aplicativo mobile com funcionalidades de e-commerce

Sobre o Projeto

O Bem Te Vi Pet-Shop Mobile é a versão mobile expandida do projeto web, desenvolvida com React Native e focada em funcionalidades de e-commerce e experiência do cliente.

O aplicativo oferece uma experiência completa de compra, agendamento de serviços e gestão de pets, integrado com sistema de pagamento para manter os clientes engajados.

Funcionalidades Principais

  • Catálogo de produtos com busca
  • Carrinho de compras e checkout
  • Funcionalidade de Pagamento (PIX)
  • Agendamento de serviços
  • Perfil do pet e histórico
  • Avaliações e reviews

Integrações

  • API do Mercado Pago
  • WebSocket para chat em tempo real
  • AWS para hospedagem
