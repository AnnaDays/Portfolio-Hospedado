Bem Te Vi Pet-Shop (Web)

Sistema de gerenciamento para Pet-Shop acessível

Sobre o Projeto

O Bem Te Vi Pet-Shop é um sistema web desenvolvido para gerenciar todas as operações de um pet shop, com foco especial em acessibilidade para administradores com deficiência visual.

O projeto foi desenvolvido com tecnologias modernas e seguindo as diretrizes de acessibilidade WCAG, garantindo que todos os usuários possam utilizar o sistema de forma independente e eficiente.

Funcionalidades Principais

  • Gestão de clientes e pets
  • Controle de estoque e produtos
  • Agendamento de serviços
  • Sistema de vendas e financeiro
  • Relatórios gerenciais
  • Design 100% acessível
  • Compatível com leitores de tela

Recursos de Acessibilidade

  • Navegação por teclado
  • Alto contraste e tamanho de fonte ajustável
  • Textos alternativos em todas as imagens
  • Estrutura semântica HTML
  • Suporte a screen readers
